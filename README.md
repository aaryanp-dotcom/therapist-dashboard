# Therapist Dashboard

A simple static therapist booking dashboard built using:
- HTML
- CSS
- Vanilla JavaScript
- Supabase (Auth + Database)

## Features
- User login
- View available therapists
- View user-specific bookings
- Supabase-powered backend (no server)

## Tech Stack
- Supabase
- Plain HTML/CSS/JS
- Static hosting compatible
